Title structure should be formatted as follows:\[Year]\_\[Brand]\_\[Set]\_\[Player First Name]\_\[Player Last Name]\_\[Card Number]\_\[Insert]\_\[Parallel]\_\[Auto | Patch | Patch Auto]\_\[Serial Number]\_\[Grading Company]\_\[Numerical Grade]\_\[Team City Name]\_\[Team Name]\_\[Rookie]The following should only be included if applicable to the card:\[Insert], \[Parallel], \[\[Auto | Patch | Patch Auto]], \[Serial Number], \[Rookie]\[Grading Company] should list Grading Company Name in their standard abbreviations (ie, PSA, SCG, BCCG, BGS, CGC, etc)\[Team City Name] should list City full name, no abbreviations\[Serial Numbers]* For eBay Titles: should be listed as /10, /25, /99, /199, etc..* For Image Renaming: should be listed as of10. of25, of99, of199, etc..* The first number set in the serial number (# before the / on the card) should only be include if meets 1 of 2 criteria: 	- first number set equals 1 (ie 1/10, 1/25, 1/99 etc..) 	- first number set equals the second number set (# after the / on the card) in the serial number (ie 10/10, 25/25, 		99/99, 199/199, etc..)Due to the 80 character limit for eBay Titles, tittle structure should follow the priority flow below: 	1. All Cards must include the following:           \[Year], \[Brand], \[Set], \[Player First Name], \[Player Last Name], \[Card Number] 	2. \[Rookie] 	3. \[Auto | Patch | Patch Auto] 	4, \[Serial Number] 	5. \[Insert] 	6, \[Parallel] 	7. \[Team Name] 	8. \[Team City Name]
